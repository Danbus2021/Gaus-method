Перед вами представлена программа для решения системы линейных уравнений методом Гаусса.
Количество уравнений ограничено числом: 100, так как при больших числах пользователю придется долго ждать вывода матрицы на экран.
Результат расчета происходит в вещественных числах с точностью до двух знаков после запятой. 
Также предусмотрено, что пользователь может прекратить процесс создания или удаления матрицы, если он будет занимать слишком много времени.
									
												Решение СЛАУ
1) После запуска программы сначала нужно выбрать количество уравнений, нажимая на ползунок.
2) Затем нажать на кнопку «Добавить СЛАУ»
3) Заполнить матрицу, нажав на «Случайное заполнение» или выбранными числами с клавиатуры по желанию пользователю
4) Применить метод Гаусса, нажав кнопку «применить метод»
5) Для просмотра решения нажать на «Ответ»
Выполнений всех действий сопровождается заполнением прогресс бара, который расположен в правом верхнем углу.
Если необходимо повторно применить метод Гаусса, то сначала необходимо Удалить СЛАУ, затем повторить 5 пунктов.
Справа от «Menu» расположена кнопка «Stop», выделенная красным. При нажатии на нее можно прервать долгий процесс создания или удаления
больших матриц.


												Меню «Файл»
1) Открыть файл – для того, чтобы открыть файл и считать оттуда значения, необходимо нажать на кнопку «Open file». 
При этом если размерность матрицы из файла и размерность СЛАУ в форме будут не совпадать, то матрица из файла не считается. 
Так же она не считается в случае, если не будет введено количество уравнений СЛАУ. Все произойдет успешно только в случае совпадения размера 
матрицы файла с размером СЛАУ в форме.
2) Сохранить файл – этой возможностью можно воспользоваться только после того, как пользователь воспользовался предыдущей кнопкой и открыл
матрицу из файла. Когда вы нажмете на «Save File» необходимо будет сохранить результаты в файл, откуда вы брали матрицу. Ответ добавиться 
в конец файла.
3) Отправить на печать – после решения СЛАУ пользователь может напечатать результаты и саму систему уравнений, либо вывести это в pdf файл. 
Для этого необходимо нажать на кнопку «Print». После нажатия на нее появится диалоговое окно, в котором нужно выбрать принтер, либо же pdf файл.
4) Журнал вычислений – при необходимости пользователь может нажать на кнопку «Calculation log» и посмотреть промежуточные 
вычисления решения СЛАУ методом Гаусса.
5) Запись в журнал вычислений - для того, чтобы записать промежуточные результаты в журнал вычислений, необходимо нажать на кнопку 
"Write calculation log". После этого откроется окно для выбора файла.
